# AI Resume Screening System

## Project Overview
This project uses AI and NLP to automatically screen and classify resumes into job categories.

## Objectives
- Automate resume screening  
- Classify resumes  
- Improve hiring efficiency  

## Dataset
Source: Kaggle  
https://www.kaggle.com/datasets/gauravduttakiit/resume-dataset  

(Note: Dataset not uploaded due to size)

## Technologies Used
- Python  
- pandas  
- numpy  
- nltk  
- scikit-learn  
- Jupyter Notebook  

## Work Done
### Week 1:
- Setup project  
- Download dataset  

### Week 2:
- Data cleaning  
- Text preprocessing  
- Stopwords removal  

## How to Run
1. Clone repo  
2. Install requirements
